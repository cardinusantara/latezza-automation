## write-tests

Gunakan **@test-writer** sub-agent untuk membuat dan menjalankan unit test setiap kali ada perubahan kode (penambahan fitur baru, perbaikan bug, refactoring, perubahan database, dll).

Sub-agent `@test-writer` sudah dikonfigurasi dengan:
- Panduan lengkap untuk backend (Jest) dan frontend (Vitest + Testing Library)
- Contoh scaffold test untuk service, endpoint, komponen React, dan utility
- Pola mocking untuk dependency eksternal (PostgreSQL, WhatsApp Baileys, Google Gemini AI, fetch API)
